# 智能垃圾分类系统 - 微信小程序

## 项目介绍

结合图像识别的环保工具，帮助用户快速识别垃圾类别，提供附近回收点导航与积分奖励。

## 功能模块

### 📷 识别模块
- **AI图像识别**：拍照识别垃圾类别
- **文字搜索**：输入垃圾名称查询
- **语音查询**：语音快速识别

### 🗺️ 导航模块
- **附近回收点定位**：地图显示周边回收站
- **路线规划**：一键导航前往
- **回收价格查询**：查看各类废品回收价格

### 🎁 积分模块
- **分类打卡**：每日签到获得积分
- **积分兑换**：兑换环保商品、话费等
- **环保任务**：完成任务获得额外积分

### 📚 科普模块
- **垃圾分类知识**：四类垃圾详细说明
- **常见误区**：常见分类错误纠正
- **环保资讯**：最新环保新闻动态

### 🏢 企业端
- **回收点入驻**：企业申请入驻平台
- **积分核销**：扫码或手动核销积分
- **数据统计**：查看经营数据报表

- **前端框架**：微信小程序原生框架

## 快速开始

### 1. 导入项目

1. 打开微信开发者工具
2. 选择「导入项目」
3. 选择 `smart-trash-sorting` 目录
4. 填写AppID（测试号也可）
5. 点击「导入」

### 2. 配置说明

- 在 `app.json` 中配置页面路由
- 在 `project.config.json` 中配置项目信息
- 在 `app.js` 中配置云开发环境ID（如需使用云开发）

#### 后端地址配置（避免写死 IP）

- 默认请求地址为 `http://localhost:3000/api`（适合开发者工具）
- 真机调试时，在开发者工具控制台执行一次：

```js
wx.setStorageSync('apiBaseUrl', 'http://你的电脑局域网IP:3000/api')
```

- 查看当前地址：

```js
wx.getStorageSync('apiBaseUrl')
```

- 恢复默认地址（localhost）：

```js
wx.removeStorageSync('apiBaseUrl')
```

### 3. 运行调试

点击开发者工具的「预览」或「真机调试」按钮即可运行。

## 课程设计文档

### 一、需求分析

#### 1.1 项目背景
随着垃圾分类政策的全面推行，居民对垃圾分类知识的需求日益增长。本系统利用AI图像识别技术，帮助用户快速准确地识别垃圾类别，提高垃圾分类效率。

#### 1.2 功能需求
- 用户能够通过多种方式（拍照、文字、语音）查询垃圾分类
- 系统能够准确定位附近回收点并提供导航
- 积分激励机制鼓励用户参与环保行动
- 企业端支持回收点管理和数据统计

### 二、系统设计

#### 2.1 架构设计
采用微信小程序原生框架，MVC架构模式：
- Model：数据层，包括垃圾分类数据库、用户数据
- View：视图层，WXML + WXSS
- Controller：逻辑层，JavaScript

#### 2.2 数据库设计
- 垃圾分类数据：本地JSON数据
- 用户数据：本地Storage存储
- 回收点数据：模拟数据（实际项目可接入地图API）

### 三、核心功能实现

#### 3.1 AI图像识别
使用微信小程序相机API拍照，模拟AI识别返回结果。实际项目中可接入腾讯云AI或百度AI图像识别服务。

#### 3.2 地图导航
使用微信小程序Map组件显示回收点位置，调用wx.openLocation实现导航。

#### 3.3 积分系统
基于本地Storage实现积分存储，包括积分增减记录、兑换历史等。

### 四、创新点

1. **多模态识别**：支持图像、文字、语音三种查询方式
2. **游戏化设计**：积分奖励机制提高用户参与度
3. **企业端支持**：完整的B端功能，支持商业运营
4. **离线可用**：核心功能不依赖网络，体验流畅
